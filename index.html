CMPN202 - Linux Server Configuration, Security, and Performance Analysis

Project Overview

This technical journal documents a comprehensive 7-week project focused on configuring, securing, and evaluating a Linux server operating system. The project demonstrates practical skills in system administration, security hardening, performance optimization, and critical analysis of operating system behavior.

Project Architecture

The project implements a dual-system architecture consisting of:

  • Server System: Ubuntu Server 24.04 LTS running headless (no GUI), accessible only via SSH
  • Workstation System: Ubuntu Desktop 24.04 LTS for administrative access and monitoring
  • Network Configuration: VirtualBox host-only network enabling secure, isolated communication

Technology Stack

Ubuntu Server 24.04 LTS Ubuntu Desktop 24.04 LTS OpenSSH UFW Firewall AppArmor fail2ban Bash Scripting Lynis nmap

Learning Outcomes Demonstrated

  • Assessment and mitigation of security vulnerabilities in operating systems
  • Proficiency with command-line tools for system administration and monitoring
  • Critical evaluation of OS design trade-offs between security, performance, and usability
  • Implementation of industry-standard security controls
  • Performance analysis and optimization using quantitative data
